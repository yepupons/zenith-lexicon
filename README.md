# Юности честное зерцало

Историко‑лингвистический проект о лексике XVIII века. Сайт объединяет фрагменты текста с подсветкой устаревших слов, словарь и полный текст произведения с оглавлением.

## Возможности
- Фрагменты текста с подсветкой архаизмов и всплывающими подсказками
- Полный текст произведения, разбитый на разделы + оглавление
- Словарь устаревших слов с краткими и полными толкованиями

## Страницы
- `Главная` — краткое описание и навигация
- `Чтение` — фрагменты с интерактивной подсветкой
- `Полный текст` — весь текст с оглавлением
- `Словарь` — словарные статьи и карточки

## Структура
- `src/pages` — страницы приложения
- `src/components` — UI‑компоненты
- `src/data/dictionary.ts` — словарь и контент чтения
- `message.txt` — исходный текст произведения

## Запуск проекта
Требуется Node.js (рекомендуется LTS).

```sh
npm i
npm run dev
```

## Сборка и превью
```sh
npm run build
npm run preview
```

## Проверки
```sh
npm run lint
npm run test
```
